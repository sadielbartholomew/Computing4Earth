Glossary and Acronyms

Glossary and Acronyms#

AMOC#

Atlantic Meridional Overturning Circulation; a large-scale system of ocean currents that transports heat, carbon, and nutrients through the Atlantic Ocean.

API#

Application Programming Interface; a defined set of rules and interfaces that allows software components to communicate with one another.

BGC#

Biogeochemistry; the study and modelling of chemical, biological, and physical processes that govern the cycling of elements such as carbon and nitrogen.

CF#

Climate and Forecast Conventions; a metadata standard that promotes consistent description and interoperability of climate and geoscience datasets.

CMIP#

Coupled Model Intercomparison Project; an international framework (a major MIP) for coordinating climate model experiments and comparing results across modelling centres.

CMOR#

Climate Model Output Rewriter; software used to convert climate model output into standardised formats and metadata conventions required by projects such as CMIP.

CPU#

Central Processing Unit; the primary processor in a computer that executes program instructions and performs general-purpose computations.

DA#

Data Assimilation; a set of techniques that combine observations with model simulations to improve estimates of a system’s state.

DSL#

Domain-Specific Language; a programming or configuration language designed for a particular application domain rather than general-purpose computing.

ENSO#

El Niño–Southern Oscillation; a coupled ocean–atmosphere climate phenomenon in the tropical Pacific that influences weather and climate worldwide.

ESM#

Earth System Model; an extension of a climate model that includes additional processes such as the carbon cycle, biogeochemistry, and interactions between the atmosphere, ocean, land, and ice.

GCM#

General Circulation Model (or Global Climate Model); a numerical model that simulates the large-scale behaviour of the atmosphere, ocean, and other components of the climate system.

GPU#

Graphics Processing Unit; a processor designed for highly parallel computations, increasingly used to accelerate scientific modelling workloads.

GRIB#

GRIdded Binary; a compact data format commonly used for storing and exchanging numerical weather prediction data.

HPC#

High-Performance Computing; the use of powerful parallel computing systems to run computationally demanding scientific simulations and analyses.

IPCC#

Intergovernmental Panel on Climate Change; the United Nations body responsible for assessing scientific knowledge related to climate change.

LES#

Large Eddy Simulation; a modelling approach that explicitly resolves large turbulent motions while parameterising smaller-scale turbulence.

MIP#

Model Intercomparison Project; a coordinated set of experiments designed to evaluate and compare the behaviour of multiple models.

MPI#

Message Passing Interface; a standard programming interface for communication between processes in parallel computing applications.

NetCDF#

Network Common Data Form; a self-describing file format widely used for storing and sharing multidimensional scientific data.

NWP#

Numerical Weather Prediction; the use of numerical models of the atmosphere to forecast future weather conditions.

OPeNDAP#

Open-source Project for a Network Data Access Protocol; a protocol that enables remote access to scientific datasets over a network.

RCM#

Regional Climate Model; a high-resolution climate model used to simulate climate processes over a limited geographical area, typically driven by boundary conditions from a GCM or ESM.

SLR#

Sea-Level Rise; the long-term increase in global or regional sea level driven primarily by ocean warming and land-ice loss.
